# 《健康地理》教案
## 第一章教案：健康地理中的空间技术——感知与描绘健康格局 (3学时)
### 第1-2讲：引言与数据获取
- 教学目标：理解健康地理空间视角的必要性；掌握GNSS/LBS与RS数据的基本原理与应用场景。
- 教学内容：
-> 课程总览：介绍三大模块逻辑关系。
-> 为什么需要空间视角？ 从约翰·斯诺霍乱地图到新冠疫情仪表盘。
-> 定位人与地点：GNSS/LBS原理，地理编码（Geocoding）实操演示（以医院、病例数据为例）。
-> 感知环境：遥感（RS）基本原理，展示NDVI、土地利用、夜间灯光等与健康相关的遥感产品。
-> 教学活动：展示不同尺度疫情地图，讨论“你从地图中看到了什么信息与可能疏漏？”

第3-4讲：数据处理、表征与核心操作

教学目标：掌握GIS核心空间操作，能进行环境与人群的空间表征。

教学内容：

GIS数据管理基础。

核心空间操作：缓冲区分析（污染源影响范围）、叠置分析（环境暴露评估）、网络分析（医疗设施可达性）。

环境/人群表征实践：如何构建“步行指数”？如何刻画社区级社会经济状态？

教学活动：使用QGIS演示“计算学校周边快餐店缓冲区”并叠置人口数据。

第5讲：空间模式与关联分析

教学目标：掌握空间插值与空间统计的基本思想，能解读其结果。

教学内容：

从点到面：空间插值（IDW，克里金法）原理与健康应用（如污染物浓度曲面）。

探测空间模式：全局与局部空间自相关（Moran‘s I, LISA），识别疾病热点/冷点。

本章总结与桥接：空间异质性、尺度效应和复杂关联如何引向对“系统复杂性”的思考？

教学活动：分析一张实际的肺癌发病率LISA聚类图，引导学生描述格局并猜测成因。

第二章教案：健康地理中的复杂系统科学——理解复杂性的思维与工具 (12学时)
第6讲：范式革命与核心概念

教学目标：实现从线性思维到系统思维的转变，掌握复杂系统核心概念。

教学内容：

引言：为何简单的空间关联不足以解释肥胖流行？

核心概念：非线性、反馈回路（正/负）、涌现、路径依赖。

系统的语言：学习绘制因果回路图。

教学活动：师生共同绘制“城市扩张-体力活动下降-肥胖率上升”的反馈回路图。

第7-8讲：建模方法I：ABM与CA

教学目标：理解ABM与CA的自下而上建模哲学，能描述其建模流程。

教学内容：

ABM原理：智能体、环境、规则。

健康地理应用：传染病传播模拟、健康行为扩散。

CA原理：规则、邻居、状态。

与GIS的桥接：如何用空间数据定义ABM的环境与智能体初始位置？

教学活动：观看NetLogo经典“病毒传播”模型运行，讨论关键参数（如接触半径）对结果（涌现的疫情曲线）的影响。

第9-10讲：建模方法II：网络分析与系统动力学

教学目标：掌握网络分析的基本度量，理解系统动力学的存量-流量思想。

教学内容：

复杂网络分析：节点、边、度、中心性、社群检测。应用于医疗协作网、疾病传播路径。

系统动力学建模：存量、流量、时间延迟。应用于区域医疗资源动态、慢性病负担模拟。

教学活动：使用Gephi可视化一个医院转诊网络，找出网络中的关键枢纽。

第11讲：模型比较与综合研讨

教学目标：能比较不同建模范式的优缺点，形成方法选型的初步判断力。

教学内容与活动：

对比ABM（个体互动）与SD（宏观流）。

分组研讨：给定“抗生素耐药性在医院的传播”问题，讨论适合采用哪种或哪几种建模方法，并简述理由。

第三章教案：健康地理中的人工智能——赋能与洞察 (9学时)
第12讲：AI引言与机器学习基础

教学目标：理解AI在健康地理中的赋能角色；掌握机器学习分类与预测任务的基本流程。

教学内容：

AI作为“加速器”与“显微镜”。

机器学习范式：监督学习。

分类任务：基于环境特征划分疾病高风险区（随机森林为例）。

预测任务：疾病发病率时空预测。

教学活动：展示同一数据集上，线性回归与随机森林预测性能的对比，引出“精度”与“解释性”的权衡。

第13讲：深度学习与计算机视觉应用

教学目标：了解CNN处理空间图像数据的能力及其在健康地理中的应用。

教学内容：

卷积神经网络（CNN）原理浅析。

应用1：街景图像识别建成环境特征（如绿化、破损）。

应用2：遥感影像多模态融合用于疾病生态位建模。

教学活动：展示通过街景图像自动识别“可步行性”并与问卷调查结果对比的研究。

第14讲：可解释AI与大语言模型

教学目标：认识“黑箱”问题的挑战与XAI的价值；了解LLM在科研辅助中的潜力。

教学内容：

可解释AI（XAI）：SHAP、LIME等方法如何帮助理解AI模型的决策。

大语言模型（LLM）：原理简介。

LLM在健康地理中的应用场景：智能文献综述、数据查询问答、报告生成、知识图谱构建。

教学活动：演示使用ChatGPT等工具辅助进行“关于城市绿地和心理健康”的中英文文献搜索与摘要提炼。

第15讲：融合、伦理与课程总结

教学目标：建立GeoAI融合观；对技术伦理进行批判性思考；完成课程整体复盘。

教学内容与活动：

融合（GeoAI）：回顾AI如何赋能空间数据提取（如CV）和复杂模型校准（如用AI优化ABM参数）。

伦理辩论：围绕“基于AI预测的精准公共卫生干预”情景，分小组讨论其可能的效益、风险（如歧视、隐私）及应对原则。

课程大总结：以一张融合三者的技术架构图，串讲完整逻辑，强调负责任的、以人为本的技术应用是终极目标。
